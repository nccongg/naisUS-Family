# Member Photos

Thư mục này chứa ảnh của các thành viên trong NaisUS Family.

## Đề xuất đặt tên file:

- `c.jpg` hoặc `c.png` - Ảnh của member C
- `n.jpg` hoặc `n.png` - Ảnh của member N  
- `h.jpg` hoặc `h.png` - Ảnh của member H

## Cách sử dụng trong code:

```tsx
// Trong MemberCard component
<Image 
  src="/images/members/c.jpg" 
  alt="Member C" 
  width={256} 
  height={256}
  className="object-cover"
/>
```
